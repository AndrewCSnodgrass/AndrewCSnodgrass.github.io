Lists Usage
Unordered List Unordered lists are generally used to present a list of items that does not necessarily need to be shown in any specific order. To start this list use the tag <li>
Ordered List Each item is shown, by default, with a number-- those lists are generally used to present a list of items that need to be shown in a specific order. To start this list use the tag <ol>
Definition List These items come in pairs-- the term to be defined and the definition usually used for glossary, vocabulary, questions and answers, etc, To start this list use the tag <dl>
  • HTML: Short for Hyper Text Markup Language a nonproprietary language interpreted by a web browser.
  • CSS: Short for Cascading Style Sheet which is used to format web pages.
  • FTP: Short for File Transfer Protocol is a network protocol for transmitting files between computers over TCP/IP
  • SSH: Short for Secure Shell or Secure Socket Shell, is a network protocol that gives users, particularly system administrators, a secure way to access a computer over an unsecure network.
